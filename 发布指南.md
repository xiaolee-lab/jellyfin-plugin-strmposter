# StrmPoster 发布指南

## 已创建的文件

- ✅ `thumb.jpg` - 插件 Logo（已存在）
- ✅ `manifest.json` - 插件清单文件
- ✅ `README.md` - 项目说明文档
- ✅ `package.ps1` - 打包脚本

## 发布步骤

### 第1步：准备 GitHub 仓库

1. 在 GitHub 创建一个新的公开仓库
2. 将项目文件推送到 GitHub
3. 确保 `thumb.jpg` 在仓库根目录

### 第2步：打包插件

#### 方式一：使用 PowerShell 脚本（推荐）

```powershell
# 在 StrmPosterForJellyfin 目录下运行
.\package.ps1
```

#### 方式二：手动打包

1. 创建 `publish` 文件夹
2. 复制以下文件进去：
   - `bin\Release\net9.0\StrmPosterForJellyfin.dll
   - `bin\Release\net9.0\StrmPosterForJellyfin.pdb` (可选)
3. 选中这些文件，右键 → 压缩为ZIP
4. 命名为 `StrmPosterForJellyfin.zip`
5. 计算 MD5 校验值：

```powershell
Get-FileHash -Algorithm MD5 StrmPosterForJellyfin.zip
```

### 第3步：创建 GitHub Release

1. 打开你的 GitHub 仓库
2. 点击右侧的 **Releases** → **Create a new release**
3. 填写信息：
   - **Tag**: `v1.0.0`
   - **Title**: `StrmPoster v1.0.0`
   - **Description**: 
     ```
     ## 功能
     - 支持 .strm 文件封面提取
     - 支持图片获取器
     - 支持计划任务
     - 支持媒体库选择
     ```
4. 上传 `StrmPosterForJellyfin.zip` 到 Release 页面
5. 点击 **Publish release**

### 第4步：更新 manifest.json

修改 `manifest.json` 中的内容：

```json
{
  "guid": "a1b2c3d4-e5f6-4a5b-8c9d-0e1f2a3b4c5d",
  "name": "StrmPoster",
  "description": "媒体库封面提取工具 - 为 .strm 流媒体文件自动提取视频关键帧作为封面图",
  "overview": "为 .strm 文件自动提取视频关键帧作为封面，支持批量提取和定时任务",
  "owner": "你的用户名",
  "category": "Metadata",
  "imageUrl": "https://raw.githubusercontent.com/你的用户名/你的仓库名/main/thumb.jpg",
  "versions": [
    {
      "version": "1.0.0.0",
      "changelog": "初始版本\n- 支持 .strm 文件封面提取\n- 支持图片获取器\n- 支持计划任务\n- 支持媒体库选择",
      "targetAbi": "10.8.0.0",
      "sourceUrl": "https://github.com/你的用户名/你的仓库名/releases/download/v1.0.0/StrmPosterForJellyfin.zip",
      "checksum": "你的MD5校验值",
      "timestamp": "2026-05-02T00:00:00Z"
    }
  ]
}
```

**需要修改的地方：**
1. `owner` - 替换为你的 GitHub 用户名
2. `imageUrl` - 替换为你的 GitHub 仓库中的 thumb.jpg
3. `sourceUrl` - 替换为你的 GitHub Release 下载地址
4. `checksum` - 填入之前计算的 MD5 校验值
5. `timestamp` - 更新为当前时间（UTC时间）

### 第5步：推送更新

将更新后的 `manifest.json` 推送到 GitHub

## 用户安装方式

### 通过存储库安装（推荐）

1. 打开 Jellyfin 管理界面
2. 进入 **插件 → 存储库**
3. 点击 **+** 新建存储库**
4. 填写：
   - **存储库名称**: `StrmPoster`
   - **存储库 URL**: `https://raw.githubusercontent.com/你的用户名/你的仓库名/main/manifest.json`
5. 点击保存
6. 进入 **目录** 标签
7. 找到 `StrmPoster` 并点击安装
8. 重启 Jellyfin

### 手动安装

1. 下载 `StrmPosterForJellyfin.dll`
2. 将文件复制到 Jellyfin 插件目录
3. 重启 Jellyfin

## 发布新版本

当需要发布新版本时：

1. 更新版本号
2. 更新 manifest.json
3. 打包新的 ZIP
4. 计算新的 MD5
5. 创建新的 GitHub Release
6. 更新 manifest.json
7. 推送更新到 GitHub

用户将在 Jellyfin 的插件目录中看到更新提示！

## 重要说明

- `targetAbi`: 10.8.0.0 表示支持 Jellyfin 10.8 及以上版本
- 如果需要支持更低版本，可修改为 10.7.0.0
- `imageUrl` 必须是可以直接访问的公开图片链接
